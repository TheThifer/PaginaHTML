Clonezilla - Restauración de disco en un equipo local

Clonezilla es un programa de clonado de disco que permite recuperar información y clonar discos para restaurarlos en otro equipo.

1- Visita la web oficial

Busca la sección "Downloads" y descarga la version stable 2.8.1-12

2- Ejecuta Rufus

Con la ISO ya descargada ejecuta rufus y selecciona las opcciones que aparecen por defecto

3.1-Pasos para crear la imagen de disco

Para probarlo deberás reiniciar el equipo y en las opciones de arranque de la BIOS poner de primera opción la unidad en la que se instaló Clonezilla.

3.2-Pasos para crear la imagen de disco

Selecciona iniciar clonecilla.

3.3-Pasos para crear la imagen de disco

Elegimos la opción Device-Image.

3.4-Pasos para crear la imagen de disco

Seleccionamos la opción usar el disco local.

3.5-Pasos para crear la imagen de disco

Omitiremos el checkeo de disco para reducir el tiempo.

3.6-Pasos para crear la imagen de disco

Seleccionamos guardar el disco local como imagen.

3.7-Pasos para crear la imagen de disco

Elegimos el disco del que se hará la imagen.

3.8-Pasos para crear la imagen de disco

Omitiremos la reparación del sistema de archivos.

3.9-Pasos para crear la imagen de disco

Comprobaremos la imagen guardada.

3.10-Pasos para crear la imagen de disco

Selecciona la opción que desee, en este caso el equipo se apagará al realizar la imagen.

3.11-Pasos para crear la imagen de disco

Dejaremos todas las opciones que aparecerán a continuación hasta lleguar a esta parte(Proceso de creación de la imagen).

3.12-Pasos para restaurar la imagen de disco

Una vez terminado el proceso encenderemos el equipo y seleccionaremos la opción Device-Image.

3.13-Pasos para restaurar la imagen de disco

Selecciona la opción usar el disco local.

3.14-Pasos para restaurar la imagen de disco

Seleccionamos la imagen creada anteriormente.

3.15-Pasos para restaurar la imagen de disco

Seleccionamos restaurar imagen a disco local.

3.16-Pasos para restaurar la imagen de disco

Elegimos el disco en el que se restaurará la imagen.

3.17-Pasos para restaurar la imagen de disco

Dejaremos las siguientes opcciones por defecto hasta llegar a esta parte (Proceso final de restauración de imagen).

Clonezilla - Restauración de disco desde un equipo diferente

En esta guia se documentará el proceso de creación y recuperación de una imagen de disco a un equipo guardando la imágen en otro de la red local. Para esto necesitaremos compartir una carpeta en la red

1.1-Pasos para crear la imagen de disco

Una vez iniciado Clonezilla elegiremos la opción Device-Image.

1.2-Pasos para crear la imagen de disco

Seleccionamos la opcción usar servidor SAMBA.

1.3-Pasos para crear la imagen de disco

Se deberá configurar la tarjeta de red. En este caso se usará una transmisión dhcp.

1.4-Pasos para crear la imagen de disco

Se deberá elegir la dirección IP y el usuario del equipo en el que se guardará la imagen.

1.5-Pasos para crear la imagen de disco

Elegimos el modo auto.

1.6-Pasos para crear la imagen de disco

Elegiremos el modo auto.

1.7-Pasos para crear la imagen de disco

Pondremos la contraseña del usuario.

1.8-Pasos para crear la imagen de disco

Elegimos el nombre que tendrá la imagen.

1.9-Pasos para crear la imagen de disco

Elegimos el disco del que se hará la imagen.

1.10-Pasos para crear la imagen de disco

Escogemos la opcción choose.

1.11-Pasos para crear la imagen de disco

Clonezilla comenzará con la creación de la imagen.

1.12-Pasos para crear la imagen de disco

Una vez terminado podremos elegier entre reiniciar o apagar el equipo.

1.13-Pasos para restaurar la imagen de disco

La imagen aparecerá en la carpeta escogida.

1.14-Pasos para restaurar la imagen de disco

Elegiremos la opcción resoredisk.

1.15-Pasos para restaurar la imagen de disco

Elegiremos el directorio donde se encuentra la imagen y la seleccionamos.

1.16-Pasos para restaurar la imagen de disco

Dejaremos las siguientes opcciones por defecto hasta que comience el proceso de restauración de la imagen.
